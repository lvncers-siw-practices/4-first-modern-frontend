## 第 3 回

[/3/](/3/)

今回の授業では、HTML・CSSだけで作った自己紹介サイトを、Reactを使って作り直していきます。

これまでのHTML・CSSでは、`index.html` に画面の構造を書き、`style.css` に見た目を書いていました。
たとえば、ページ上部には `header`、メインの内容には `main`、プロフィール紹介には `section`、ページ下部には `footer` を使って、1つのWebページを作ってきました。

Reactでも、最終的にブラウザに表示される画面はHTMLとCSSで作られます。
しかし、コードを書く場所や考え方が少し変わります。
Reactでは、HTMLファイルに直接画面を書くのではなく、`App.tsx` やコンポーネントファイルの中に、JSXという書き方で画面を書きます。

今回のゴールは、完成コードを写すことではありません。
HTML・CSSで作った画面をReactに移し替える過程を通して、Reactの基本的な考え方を理解することです。
